Классы для работы с переносом или длиной текста

nowrap,whitespace-normal, break-word, overflow-ellipsis, overflow-hidden, overflowX-auto, overflowY-auto

только для мобильных: nowrap__mobile, whitespace-normal__mobile, break-word__mobile, overflow-hidden__mobile

Классы для выравнивание текста в контейнере по горизонтали

text-left, text-right, text-center, text-justify

Постфикс __mobile дописанный к этим классам устанавливают такие значения только для мобильных устройств.

{% highlight html %}
text-left
text-center text-left__mobile
text-right
{% endhighlight html %}
text-left
text-center text-left__mobile
text-right

Классы для выравнивание текста в контейнере по вертикали

va-middle, va-top, va-bottom

Классы для изменения регистра текста

text-lowercase, text-uppercase, text-capitalize

Постфикс __mobile дописанный к этим классам устанавливают такие значения только для мобильных устройств.

text-capitalize

Классы для изменения толщины текста

text-regular, text-bold,

Постфикс __mobile дописанный к этим классам устанавливают такие значения только для мобильных устройств.

Классы для изменения размера текста

Следующие классы устанавливают размер шрифта в rem эквивалентное числу в классе, указанного px: fz{FZ},

где FZ = 10, 11, 12, 13, 14, 16, 18, 20, 24, 48

Постфикс __mobile дописанный к этим классам устанавливают такие значения только для мобильных устройств.

Классы для изменения междустрочного интервала

Следующие классы устанавливают значение в px равное числу в классе, указанного px: lh{LH},

где LH = 10, 12, 14, 16, 18, 20, 24, 48

Постфикс __mobile дописанный к этим классам устанавливают такие значения только для мобильных устройств.

{% highlight html %}
fz10 = 10px lh14 = 14px
fz14 = 14px lh16 = 16px, на мобильном lh18__mobile = 18px
fz18 = 18px lh20 = 20px
fz20 = 20px lh20 = 20px, fz24__mobile = 24 на мобильных lh24 = 24px
fz24, lh24 = 24px
{% endhighlight html %}
fz10 = 10px lh14 = 14px
fz14 = 14px lh16 = 16px, на мобильном lh18__mobile = 18px
fz18 = 18px lh20 = 20px
fz20 = 20px lh20 = 20px, fz24__mobile = 24 на мобильных lh24 = 24px
fz24 = 24px, lh24 = 24px

Классы изменения цвета текста

Доступные для использования классы:

{% for item in site.data.jekyll.brandnamecolor %} text-{{ item.name }}, {% endfor %}

Классы для изменения цвета текста при наведении (hover)

{% for item in site.data.jekyll.brandnamecolor %} textHover-{{ item.name }}, {% endfor %}

Для активаци hover-эффекта текста при наведении на элемент-родителя выставить класс на родителе - textHoverTarget

{% highlight html %}
...
text-primary textHover-danger
text-white textHover-grey-light
text-clear bg-muted text-center textHover-success on parent
{% endhighlight %}
{% for item in site.data.jekyll.brandnamecolor %} {% if item.name == 'primary' %}
text-{{ item.name }} textHover-danger
{% elsif item.name == 'white' %}
text-{{ item.name }} textHover-grey-light
{% elsif item.name == 'clear' %}
text-clear bg-muted text-center textHover-success on parent
{% else %}
text-{{ item.name }}
{% endif %} {% endfor %}
