Внешние отступы

Классы формируются по схеме: отступ сверху mt-{X}, справа mr-{X}, снизу mb-{X}, слева ml-{X}. Отступ со всех сторон margin-{X}

X может принимать следующие значения: {% for item in site.data.jekyll.marginsizes %} {{ item.name }}, {% endfor %} auto

Для мобильных устройств к правилу дописывается постфикс __mobile.

Пример: ml-10, margin-20, ml-0__mobile, mr-20, mb-10



Внутренние отступы

Классы формируются по схеме: отступ сверху pt-{X}, справа pr-{X}, снизу pb-{X}, слева pl-{X}. Отступ со всех сторон padding-{X}

Для мобильных устройств к правилу дописывается постфикс __mobile.

X может принимать следующие значения: {% for item in site.data.jekyll.paddingsizes %} {{ item.name }}, {% endfor %}

Пример: pl-10, padding-10, pl-0__mobile, pr-20, pb-10



GAP

Классы формируются по схеме: gap-{X}

Для мобильных устройств к правилу дописывается постфикс __mobile.

X может принимать следующие значения: {% for item in site.data.jekyll.gapsizes %} {{ item.name }}, {% endfor %}

Пример: gap-10, gap-5__mobile
