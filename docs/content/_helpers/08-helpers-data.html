Вывод или выбор цвета

colorField - родитель

colorField_formControl - цветной кружок - заполненый бекграундом переданным через style="background:..."

Блоки с данными для фильтрации

square - базовый класс оформления блоков

При наличии в блоке класса `isHover`, `active` или при наведение по псевдоклассу будет изменяться фон также как и при hover

square_{цвет}__onAction - цвет фона при наведении, при .actvie или .isHover

square__borderThick - бордер толщиной 2px

square-title - блок с заголовком

square-text - блок с данными

square-text_ValueSmall - модификатор с дополнтильным данными

square-action - блок с кнопкой действия
