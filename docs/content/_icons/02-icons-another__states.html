Модификация иконки bicolors-plus

Использование одной иконки bicolors-plus для cкрипта collapse. И создание эффекта замены на `minus` при расскрытии списка.
Для этого используется класс `.limited`.

Модификация иконки bicolors-favorite, bicolors-star

Использование иконки `bicolors-favorite` для применения с классом родителя `favorite__active` и создание эффекта замены на `bicolors-favorite-full`.

Использование иконки `bicolors-star` для применения с классом родителя `star__active` и создание эффекта замены на `bicolors-star-full`.
