Для вывода строк таблицы в виде дерева - со скрытием подстрок исопльзуем скрипт jQuery treetable. Для этого таблице дописываем класс `js-tree-table` и инициализируем скрипт с помощью initTreeTable(); При необходимости делаем import {initTreeTable} from "../utils";
