Использование иконок в шапке.

Для подсказки вставить контейнер с иконкой и текстом подсказки. На ячейку добавляем класс tableHelpIcon_td

Для функционала раскрытия колонок используем . На ячейку добавляем класс collapseTdIcon_td

Иконки в теле таблицы

Обычно функционал с ионками располагается в последней ячейки строки таблицы. В этом случае на ячейку навешиваем классы text-right nowrap

Для кнопок используем класс btn-icon
