Для выравнивания данных в ячейках используем классы text-center, text-right на нужные ячейки.

Добавление границ в ячейках используем класс модификатор табилцыtable__border.

Изменение толщины линии ячейки добавляет соответсвующий класс border-top__thick, border-bottom__thick, border-left__thick или border-right__thick

Убрать границу можно с помощью классов border__none, border-top__none, border-bottom__none, border-left__none, border-right__none

Для черезстрочного изменения фона используем классы-модификаторы таблицы table__striped и table__stripedReverse (изменена очередность чредования фона). Устаревший класс - table-striped

Миниимальну ширину для первой ячейки каждой строки можно установить через класс-модификатор табилцыtable__firstChildMinWidth
