Лёгкий vanilla-JS скрипт (ES-модуль) для управления сворачиванием/разворачиванием групп ячеек таблицы с автоматическим пересчётом `colspan`.

Возможности

Сворачивание/разворачивание строк по клику на триггер

Автоматический пересчёт colspan у ячеек выше триггера, у родителя триггера и у ячеек ниже (в ручном режиме)

Поддержка нескольких групп сворачивания на одной странице

Один элемент может реагировать сразу на несколько групп

Ручной и автоматический режимы изменения colspan (сложение/вычитание)

Корректная работа с colspan у самих скрытых ячеек

Делегирование событий — работает с динамически добавленными элементами

Встроенное логирование и обработка ошибок

Полностью настраиваемые классы и data-атрибуты

Data-атрибуты

У триггера (кнопка, ссылка и т.д.)

data-collapse-target-get - Обязательный атрибут, Имя группы, которую нужно свернуть/развернуть

У цели (строка или ячейка /)

data-collapse-target-set - Обязательный атрибут, Имя группы (можно несколько через пробел/запятую)

У ячеек с пересчётом colspan

data-collapse-colspan-set - Обязательный атрибут, Имя группы, на которую реагирует ячейка (можно несколько)

data-collapse-colspan-value - Необязательный атрибут, Ручной режим: конкретное число, на которое изменится colspan (может быть отрицательным)

data-collapse-colspan-calc - Необязательный атрибут, Авто-режим: plus / add — сложение, любое другое значение (или пусто) — вычитание

Приоритет: если указаны оба атрибута (value и calc), используется ручной режим (value).

Как работает пересчёт

Родитель триггера ( или , в котором лежит кнопка)

По умолчанию: вычитание

Скрипт находит следующую строку после строки с триггером и суммирует colspan ячеек с нужной группой

Можно переопределить через data-collapse-colspan-calc="plus"

Ячейка ВЫШЕ триггера

Работает только в автоматическом режиме

Режим задаётся через data-collapse-colspan-calc (plus/add — сложение, иначе — вычитание)

Считает скрытые ячейки в следующей строке после каждого триггера группы

Ячейка НИЖЕ триггера

Работает только в ручном режиме (data-collapse-colspan-value)

Без этого атрибута colspan не меняется
