Скрипт для автоматизации копирования текста в буфер обмена по клику на элементы интерфейса.

  • Защита от повторной инициализации
  • Поддержка модальных окон
  • Интеграция с уведомлениями. Автоматический показ toast-уведомлений об успехе или ошибке через $.notify. (jQuery Notify)

Скрипт необходимо инициализировать один раз при загрузке страницы initCopyDataAttrToClipboardBtns() import {initCopyDataAttrToClipboardBtns} from "./copyTextToClipboardNotify"

Доступные data-* атрибуты

data-copy - Обязательный. Текст для копирования. Может быть строкой или функцией, возвращающей Promise.

data-copy-success-message-text - Текст уведомления при успешном копировании. По умолчанию "Скопировано"

data-copy-success-message-type - Тип (цвет) уведомления об успехе (например, success, info). По умолчанию - "success"

data-copy-error-message-text - Текст уведомления при ошибке копирования. По умолчанию - "Ошибка"

data-copy-error-message-type - Тип уведомления об ошибке (например, danger, warning). По умолчанию - "danger"
