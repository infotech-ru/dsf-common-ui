Легковесный скрипт на Vanilla JS для автоматического подстраивания высоты текстовой области (textarea) под объем введенного текста.

  • Инициализация может быть проведена как для всего документа, так и для конкретного DOM-элемента или контейнера
  • Поддержка задания минимальной (minHeight) и максимальной (maxHeight) высоты. При превышении максимума появится стандартный скроллбар.
  • Защита от повторной инициализации:
  • К каждому инициализированному элементу добавляется метод updateAutoresize(), позволяющий принудительно пересчитать высот
  • Возможность передать функцию onResize, которая будет вызываться при каждом изменении высоты элемента
  • Для корректного определения новой высоты скрипт сначала сбрасывает высоту элемента до 28px, а затем считывает scrollHeight. Это необходимо, чтобы высота уменьшалась при удалении текста. Значение 28px используется как базовое для сброса (обычно соответствует высоте одной строки с учетом padding и border в вашей CSS-сетке)
{% highlight js %} import { AutoresizeTextarea } from './autoresizeTextarea'; // Базовая инициализация для всех textarea с классом .js-formControl__resize на странице AutoresizeTextarea(); // Или с настройками AutoresizeTextarea({ selector: '.my-custom-textarea', context: '#modal-window', maxHeight: 200 }); {% endhighlight %}
Параметр Тип По умолчанию Описание
selector string '.js-formControl__resize' CSS-селектор для поиска текстовых областей.
context string | Element | Document document Контекст, внутри которого осуществляется поиск элементов. Может быть DOM-элементом, объектом document или строкой-селектором (например, '#modal').
minHeight number | null null Минимальная высота в пикселях. Высота никогда не будет меньше этого значения.
maxHeight number | null null Максимальная высота в пикселях. При превышении этой высоты у элемента появится вертикальный скролл (overflow-y: auto).
onResize function | null null Callback-функция, вызываемая после изменения высоты. Принимает аргументы: (textareaElement, newHeight).
