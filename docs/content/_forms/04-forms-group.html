Класс обертка - inputGroup

Класс-контейнер для элементов формы - селектов, инпутов - form-control-container. Каждрый инпут, селект в своем контейнере form-control-container

Класс-контейнер для элементов спереди - inputGroup_prepend

Класс-контейнер для элементов сзади - inputGroup_append

Класс для визульного отображении вне полей в классах-контейнерах (inputGroup_prepend, inputGroup_append) - inputGroup_outer

Класс для визульного отображении внутри полей в классах-контейнерах (inputGroup_prepend, inputGroup_append) - inputGroup_inner

Для кнопок-иконок (btn-icon) вне полей (inputGroup_outer) добавляется класс-модификатор - btn-icon__form
