Используется скрипт bootstap-select

Смещение выпадающего списка влево.

1. 'data-dropdown-align-right' => 'auto' - автоматическое смещение

2. 'class' => 'dropdown-menu-right' - принудительное смещение

{% highlight php %} ... widget(Select::class, [ 'options' => [ 'class' => 'dropdown-menu-right', или 'data' => [ 'dropdown-align-right' => 'auto', ]) {% endhighlight %}

Dropup menu - Открытие выпадающего списка вверх

dropupAuto - по умолчанию true, т.е автоматические выбирает, когда открывать вверх. Если выставить false и добавить класс dropup - будет всегда открывать вверх

Максимальное количество выбираемых пунктов при мультивыборе

'data-multiple' => true

Ограничьте количество пунктов, которые могут быть выбраны с помощью атрибута data-max-options. Он также работает для групп опций.

Поиск внутри выпадающего меню

Включается 'data-live-search' => true

Форматирование текста выбранного пункта

*value - Список выбранных значений, разделенный запятыми (по умолчанию)

*count 'data-selected-text-format' => '2' - Если выбран один элемент, то отображается значение пункта. Если выбрано более одного, то отображается количество выбранных элементов, например 2 из 6 выбрано

*count > x 'data-selected-text-format' => 'count > 2', где x - количество элементов, выбранных при изменении формата отображения со `*value` на `*count`.

* static 'data-selected-text-format' => 'static' Всегда отображайте заголовок выбора (placeholder), независимо от выбора

Стилизация отдельных пунктов в выпадающем меню

Классы и стили, добавленные к пункту, переносятся в поле выбора. Используются предустановленные классы изменения цвета и фона

2. Используя атрибут data-width

'data-width' => 'auto' - автоматическая подсторйка пол максимальную ширину.

'data-width' => 'fit' - автоматически подстраивает ширину селекта под ширину выбранной в данный момент пункта

{% highlight php %} ... widget(Select::class, [ 'options' => [ 'data' => [ 'width' => 'auto', 'width' => 'fit', 'width' => '100px', 'width' => '75%', ]) {% endhighlight %}

Размер по высоте выпадющего меню.

По умолчанию для опции 'data-size' => 'auto' - меню всегда раскрывается, чтобы показать столько пунктов, сколько позволяет окно, не обрезая их.

'data-size' => false - будет показывать все пункты.

'data-size' => '5' - устанавливает максимальное количество видимых пунктов в меню.

DOM: Вывод выпадающего меню в/вне контейнера родителя

По умолчанию 'data-container' => 'body'

Это полезно, если элемент select находится внутри элемента с overflow: hidden. и подменю находится в самом низу около закрывающегося тега body

Но можно выводить и непосредствено в контейнере родителя селекта 'data-container' => false или в любом другом месте 'data-container' => '.other-container'
