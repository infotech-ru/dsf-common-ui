Для всплывающих контейнеров используются скрипт tooltip.js и его расширенный вариант - popover.js

Класс для увеличения максимальной ширины popover `.popover__wide`
Класс для убирания максимальной ширины popover `.popover__nonmax`

https://getbootstrap.com/docs/4.6/components/tooltips/
https://getbootstrap.com/docs/4.6/components/popovers/
