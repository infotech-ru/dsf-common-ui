progress - класс обертка. Возможно совмещение нескольких прогрессбаров через эту общую обретку

progress__textOuter - класс модификатор обертки. Выносит значенени прогрессбаров наружу.

progress__borderRadius - класс модификатор обертки. Создает закругление при применении на обертке классов borderRadius-* или указания у переменной --progress-border-radius значения.

--progress_height - высоты прогреccбара задаваемая через переменную в style у обертки, по умолчанию 14px

progress-bar-success, progress-bar-info, progress-bar-default, progress-bar-danger, progress-bar-warning, progress-bar-primary - базовые цвета у прогресбаров задаваемые через классы.

bg-* - дополнительные цвета у прогресбаров задаваемые через класс-хелпер

progress-bar-striped - класс-модифиакатор прогресбара создающий прогрессбар "Зебра"

progress-bar-animated - класс-модифиакатор прогресбара создающий анимацию "Зебры"

Прогрессбар-кольцо

Через style в progress-ring-bar могут использоваться переменные:

--progress-ring_percent - прогресс заполняемой части кольца, по умолчанию 0

--progress-ring_border - толщина кольца, по умолчанию 8px

--progress-ring_size - размер кольца, по умолчанию 100%

--progress-ring_bg - цвет внутреннего фона, по умолчанию белый

--progress-ring_stroke - цвет кольца, по умолчанию grey2

--progress-ring_fill - цвет заполняемой части кольца, по умолчанию light-blue
