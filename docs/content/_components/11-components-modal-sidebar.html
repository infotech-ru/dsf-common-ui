Виджет ModalSidebar показывает боковое окно слева или справа экрана. Создан на основе bootstrap modal и использует modal.js.

Для правильной отрисовки бокового окна нужно дополнить вызов модалки параметром: sidebar: true.

Размер окна изменяется в size: SIZE_LARGE и SIZE_SMALL.

{% highlight html %} 'size': SIZE_LARGE 'size': SIZE_SMALL {% endhighlight %}

Сторона открытия изменяется через side: SIDE_LEFT и SIDE_RIGHT.

{% highlight html %} 'side': SIDE_RIGHT 'side': SIDE_RIGHT {% endhighlight %}

Шапка окна отрисовывается через в header исплоьзуя для заголовка h4 классы h4 modal-sidebar-headerTitle.

Рядом могут находится другие элементы - label, icon...

{% highlight html %} php: use app\widgets\ModalSidebar; ModalSidebar::begin( [ 'header' => '

' . Yii::t('app', 'Заголовок') . '

', 'size' => ModalSidebar::SIZE_LARGE, 'side' => ModalSidebar::SIDE_LEFT, ] ... ) ModalSidebar::end() js: show.call(this, { sidebar: true, ... }) {% endhighlight %}
