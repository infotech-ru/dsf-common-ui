Виджет Modal показывает модальное окно посредине экрана.

Используемый скрипт bootstrap v3 modal.js

Размер окна изменяется добавлением класса.
Для классов modal-lg и modal-sm в yii можно использовать SIZE_LARGE и SIZE_SMALL

{% highlight html %} 'size' => SIZE_LARGE 'size' => SIZE_SMALL 'size' => 'modal-xl' {% endhighlight %}

modal-xs (width: 320px)

modal-sm (width: 700px)

modal-md (width: 900px)

modal-lg (width: 1200px)

modal-xl (width: 1240px)

modal-xxl (width: 1320px)

modal-xxxl (width: 1400px)

{% highlight html %} 'size' => 'modal-xl modal-xxl', {% endhighlight %} Класс modal-responsive - добавления резиновой ширины модалки. {% highlight html %} 'size' => 'modal-responsive', {% endhighlight %} Класс modalScrollableBody - Добавление вертикального скрола для modal-body. {% highlight html %} 'size' => 'modalScrollableBody' {% endhighlight %} Шапка окна отрисовывается через 'header'. {% highlight html %} 'header' => '
Icons::wrapper('bicolors-checklists__24vb', ['class' => 'svg--icon svg--icon__24vb modalHeader_itemTitleSvg'])

' . Yii::t('app', 'Название') . '

Icons::withText('bicolors-timeback', Yii::t('app', 'История изменения'))
', {% endhighlight %} Рядом с названиеи могут находится другие элементы - иконки, кнопки, лейблы.
