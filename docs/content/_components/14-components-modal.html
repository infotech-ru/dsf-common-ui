--- Title: Модальное окно menutitle: Модальное окно (modal) category: components anchor: modal-modal order: 14 --- Виджет Modal показывает модальное окно посредине экрана. Используемый скрипт bootstrap v3 modal.js Размер окна изменяется через добавление класса: 'size' => SIZE_LARGE и SIZE_SMALL (это modal-lg и modal-sm) или просто класс modal-xs (width: 320px) modal-sm (width: 700px) modal-md (width: 900px) modal-lg (width: 1200px) modal-xl (width: 1240px) modal-xxl (width: 1320px) modal-xxxl (width: 1400px) 'size' => 'modal-xl modal-xxl', `modal-responsive` - класс для добавления резиновой ширины модалки 'size' => 'modal-responsive', Добавление вертикального скрола осуществляется добавлением класса `modalScrollableBody` - 'size' => 'modalScrollableBody' Шапка окна отрисовывается через 'header'. {% highlight html %} 'header' => '
' . Icons::icon('bicolors-list_questionnaires__24vb') . '

' . Yii::t('app', 'Название') . '

'' . Icons::icon('bicolors-timeback') . ' История изменения',
', {% endhighlight %} Рядом с названиеи могут находится другие элементы - иконки, кнопки, лейблы.
