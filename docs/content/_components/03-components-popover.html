Для всплывающих контейнеров используются скрипт tooltip.js и его расширенный вариант - popover.js

Класс для увеличения максимальной ширины popover .popover__wide

Класс для убирания максимальной ширины popover .popover__nonmax

Класс для увеличения максимальной ширины tooltip .tooltip__wide

Класс для убирания максимальной ширины tooltip .tooltip__nonmax

Класс для выравнивания содержимого по левому краю в tooltip .tooltip__textLeft

JS: popover({ options: { placement: 'auto', sanitize: false, customClass: 'popover__nonmax', }, clickBody: true, })
