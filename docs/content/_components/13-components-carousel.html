Используется скрипт bootstrap v4 carousel.js

Для vue есть компонент Carousel.vue

Классы для изменения пропорции картинок с 4/3 на 16/9 и 16/10carousel__aspectRatio16-9 carousel__aspectRatio16-10

Класс carousel-indicators__line - изменяет превью из картинок в чёрточки

Класс carousel-indicators__bigPreview - изменяет размер превью картинок в больший размер

Класс carousel-action carousel-action_visible - вывод кнопок действия без эффекта наведения

Класс carousel-action__bottom - объединение вывода кнопок действия внизу

Класс carousel-action__top - объединение вывода кнопок действия сверху

Класс carousel-action__bottomRight - вывод кнопок действия внизу справа

Класс carousel-action__bottomLeft - вывод кнопок действия внизу слева

Класс carousel-action__topRight - вывод кнопок действия вверху справа

Класс carousel-action__topLeft - вывод кнопок действия вверху слева

Класс carousel-action__center - вывод кнопок посередине
