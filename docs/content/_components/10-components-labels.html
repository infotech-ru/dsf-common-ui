--- Title: Labels menutitle: Labels category: components anchor: components-labels order: 10 --- Для `.label` используются следующие цветовые модификаци: 'default', 'primary', 'success', 'danger', 'warning', 'info', 'yellow', 'black', 'dark-grey' {% highlight html %} label label-default label label-primary label label-success label label-danger label label-warning label label-info label label-yellow label label-black label label-dark-grey {% endhighlight %}
label label-default label label-primary label label-success label label-danger label label-warning label label-info label label-yellow label label-black label label-dark-grey

Для вывода дополнительной информации в виде `label`, но без модификации всего контейнера, на котором навешивается класс - используется `.label__after` или `label__before` и необходимый модификатор цвета.
Текст дополнительного поля выводится через дата-атрибут `data-label-text`.
Возможно изменения отступа от поля через изменения в style переменной --label_margin-left или --label_margin-right
