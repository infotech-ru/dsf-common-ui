Комментарии имееют следующие оформление.

Блок с вводом сообщения как правило ставиться как перед списком комментариев, так и после

Цвет фона списка комментариев задается с помощью helper-классов. По умолчанию, bg-light-yellow

Если есть возможность прикреплять файлы, и существует информационое поле с именем добавленяемого файла, то высота списка должна быть переключаемой: + высоту информационного поля. Например с 240 на 264. А само поле менять цвет с цвета фона списка комментариева (по умолчанию bg-light-yellow) на цветполя ввода комментариев - bg-white

Тень на блоке одиночного комментария задается через commentBlock_comments__shadow

Цвет фона блока одиночного комментария задается с помощью helper-классов. По умолчанию используется bg-light-green или bg-info

Для увеличенного поля ввода используется класс commentBlock_inputGroup__bigHeight

Для возможности изменения высоты textarea выставляем класс commentBlock_formControl__resizeVert на ней.

Для появления полосы прокруткии на поле ввода на textarea указываем класс overflowY-auto.
