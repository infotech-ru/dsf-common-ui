Наши иконки формируются в спрайт.

Исходники для спрайта иконок грузятся в проект-пакет `dsf-common-ui` в папку `svg-for-sprite`.

Собирается спрайт локально, запуском команды `grunt`.

Обновленная версия попадает в папки проект-пакета и выгружается в github.

Важно! Исходники иконок перед загрузкой нужно обработать, чтобы viewBox совпадал с контурами иконки и имел размер 24px по высоте.
